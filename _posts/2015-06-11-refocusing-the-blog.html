I have decided it is time for a refocus of this blog. I am going to post more often on a more focused set of topics. I have been interested in the new Roslyn code analysis features that are coming in Visual Studio 2015, so going forward you will see a lot more posts on topics surrounding code analysis and you will see them on a more regular basis. I have started contributing to the open source Style Cop Analyzers project, as I feel that is a great way to learn more about this technology and I will share as much as I can on this blog.

To fully reap the benefits of this refocus, I have been following the methodology laid out by John Sonmez in his blogging course and I feel it provides simple yet useful guidance on how to properly manage your blog. Heck, I even bought his book. If you haven't checked out the free blogging course, I encourage you to do so, maybe it will provide you the kick in the butt you need to get out there.
